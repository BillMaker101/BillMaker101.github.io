Your Own Chess Code

graphical user interface - to - state-of-the-art chess engine

designed in 2024 by author of 1980's chess program 'Sfinks'

This project holds the Microsoft Word documents which comprise ChessCode. ChessCode is a step-by-step process for creating a state-of-the-art chess program in Microsoft Visual Studio. ChessCode was designed to run on Intel 64-bit CPU's only. It uses Intel intrinsic instructions so may not run on other CPU's. It is written in 'C' with a little 'C++'.

The text in purple, "ChessCode Readme", is a link to the ChessCode files. Click on it to access the files. You will see "Releases" on the right. Click on it to download the latest Release (.zip or .tar.gz) file to get your copy of the 8 chapter book. Enjoy building your own chess game as seen above.
